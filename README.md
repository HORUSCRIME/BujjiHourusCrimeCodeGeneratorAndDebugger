## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js, npm, and Python installed on your system. You will also need an API key from Google AI Studio.

* **npm**
    ```sh
    npm install npm@latest -g
    ```
* **Google Gemini API Key**
    Get your key here: [Google AI Studio](https://aistudio.google.com/app/apikey)

### Installation

1.  **Clone the repo**
    ```sh
    git clone [https://github.com/your-username/your-repo-name.git](https://github.com/your-username/your-repo-name.git)
    cd your-repo-name
    ```

2.  **Backend Setup**
    ```sh
    cd backend
    pip install -r requirements.txt
    ```
    Create a `.env` file in the `backend` directory and add your API key:
    ```
    GEMINI_API_KEY='YOUR_API_KEY_HERE'
    ```

3.  **Frontend Setup**
    ```sh
    cd ../frontend
    npm install
    ```

## Usage

1.  **Run the Backend Server**
    Navigate to the `backend` directory and run:
    ```sh
    python app.py
    ```

2.  **Run the Frontend Development Server**
    In a new terminal, navigate to the `frontend` directory and run:
    ```sh
    npm start
    ```

3.  Open your browser and go to `http://localhost:3000` to start using the application!
